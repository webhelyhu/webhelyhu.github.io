Hangman game

Egy angol nyelvű szókitaláló
- Egyszerű Javascript
- API kezelés három végpontról
- IP cím alapján helymeghatározás

Sajátmese.hu WP weboldal

Wordpress weboldal meseterápiával foglalkozó házaspár számára.

- Nyitóoldalon egyéni CSS animáció
- Docker konténerbe költöztetés

YourPlaces

Felhasználók saját helyeiket rögzíthetik, mindenki látja a felhasználók bejegyzéseit. Maximilian anyaga alapján.
- MongoDB adatbázis, Mongoose kapcsolat
- Bárki létrehozhat felhasználót
- Szerveroldali authentikáció, JWT+bcrypt
- Docker konténerben

NodeJS Chat

Chat rendszer. Beléphetsz az általad megadott nevű szobába, ott írhattok közös üzeneteket.
- Socket.IO használata
- Szerveroldali bad-words szűrő
- Helyadatok küldése
- Heroku.com aloldalon publikálva

Pumpkin2

- Egyéni táblázat (méretfüggő oszloptartalom)
- Szerkesztő külön vagy oldalon belül
- Hooks és Redux változat

Időpontfoglaló rendszer

Egy nyílt napon a terapeuták előre megadott időpontokban fogadnak ügyfeleket. Előfordul, hogy több terapeuta is fogad ugyanabban a témában ügyfeleket. Minden időpontot csak az időpont előtti három órában lehet foglalni.
- MongoDB, Mongoose, NodeJS backend
- Közvetlen html jelentkező oldal, két megjelenítéssel, extra rövid válaszidővel (<20msec)
- React vezérlőpult az asszisztenseknek
